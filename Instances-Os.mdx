## Key Features

- **One-Click OS Installation**\
  Install or switch the operating system with minimal effort.
- **Multiple OS Options**\
  Support for widely used Linux distributions and Windows Server editions.
- **Fast OS Switching**\
  Redeploy instances quickly without manual setup.
- **Automated Installation**\
  The installation process is fully automated to reduce errors and downtime.

---

## How OS Installation Works

The OS installation process follows these steps:

1. **Select OS**\
   Choose the required operating system from the dropdown in the instance control panel.
2. **Initiate Installation**\
   Click the **Install OS** button to start the process.
3. **Monitor Progress**\
   Track installation status directly from the control panel.
4. **Completion**\
   Once completed, the instance will boot with the newly installed operating system.

## Installing or Reinstalling an OS

To install or change the operating system:

1. Log in to the **HynixCloud Dashboard**
2. Navigate to **Instances**
3. Select the instance you want to modify
4. Choose the required OS from the dropdown menu
5. Click **Install OS**
6. Monitor installation progress from the dashboard

Once the process is complete, the instance will be available with the selected operating system.

> **Note:** Reinstalling the OS may overwrite existing data. Ensure backups are taken before proceeding.

## Access Credentials

### Credential Delivery

- After OS installation, **access credentials are sent via email**
- Please check your inbox and spam folder if not received immediately

---

### Credential Management

You can update instance credentials from the dashboard:

1. Go to **Instances**
2. Select the required instance
3. Click **Change Password**

---

### Default Usernames

- **Windows Server:** `Administrator`
- **Linux (Ubuntu / CentOS):** `hynix` _(or system default user)_

> Password changes take effect immediately after confirmation.
